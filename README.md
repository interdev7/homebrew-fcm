# 🍺 Homebrew Tap for FCM CLI

Official Homebrew tap for installing **FCM CLI** — a production-ready Firebase Cloud Messaging tool written in Go.

---

## 🚀 Install

```bash
brew tap interdev7/fcm
brew install fcm
```

---

## 🔄 Upgrade

```bash
brew update
brew upgrade fcm
```

---

## 🧹 Uninstall

```bash
brew uninstall fcm
```

---

## 📦 About FCM CLI

FCM CLI is a fast and production-ready tool for sending Firebase Cloud Messaging (FCM) notifications.

👉 Features:

* Send to single device, multiple tokens, topic or condition
* Parallel multicast with progress bar
* Retry with exponential backoff
* Structured logging (color + JSON)
